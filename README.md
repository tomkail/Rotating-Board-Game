# Ring Board Game

A prototype board game with a rotating ring mechanic where players place tiles and trigger effects.

## Running the App

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation & Setup

1. Navigate to the prototype directory:

```bash
cd prototype
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and go to the URL shown in the terminal (typically `http://localhost:5173`)

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Game Features

- Select number of players (1-8)
- Place different tile types on the board
- Connect tiles to form groups
- Rotate tile groups or the entire board
- Trigger tile effects when players land on them
- Save and load game states
